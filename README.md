# Exercice 1 : Changement de thème (Dark / Light Mode)
Créer une page avec un bouton Toggle Thème.
- Au clic, basculer entre un thème clair et un thème sombre en ajoutant/retirant une
classe CSS sur le .
- Mémoriser le choix dans localStorage pour le restaurer au rechargement

# Exercice 2 : Barre de progression
Créer une barre de progression (0 % → 100 %) avec deux boutons : +10% et Réinitialiser.
- La largeur d'un coloré doit refléter la valeur courante.
- Afficher le pourcentage en texte à l'intérieur de la barre.
- Empêcher la valeur de dépasser 100 %.

# Exercice 4 :Délégation d'événements
Créer dynamiquement une liste de 5 éléments avec un bouton Ajouter.
- Attacher un seul écouteur d'événement sur le <ul> parent (pas sur chaque <li>).
- Au clic sur un <li>, changer sa couleur de fond
- Les nouveaux éléments ajoutés doivent aussi réagir au clic sans ajouter de nouvel
écouteur.

# Exercice 5 : Accordion / FAQ
Créer une section FAQ avec 5 questions/réponses sous forme d'accordéon.
- Au clic sur une question, afficher ou masquer la réponse correspondante.
- Une seule réponse peut être ouverte à la fois (fermer l'ancienne à l'ouverture d'une
nouvelle).
- Ajouter une animation CSS de transition sur l'ouverture/fermeture.

# Exercice 6 : Validation de formulaire
Créer un formulaire d'inscription avec les champs : Nom, Email, Mot de passe, Confirmation.
- Valider chaque champ à la soumission : aucun champ vide, email valide (avec @),
mot de passe ≥ 8 caractères, les deux mots de passe identiques.
- Afficher les messages d'erreur directement sous chaque champ concerné.
- En cas de succès, afficher un message de confirmation et réinitialiser le formulaire.
